# customer-churn-analysis
## Project Overview

This project analyzes customer churn patterns in a telecom company using SQL and Power BI. The objective is to identify key churn drivers, understand customer behavior, and recommend retention strategies.

## Business Problem

Customer churn directly impacts revenue and profitability. Understanding why customers leave helps businesses improve retention and customer satisfaction.

## Dataset

* Total Records: 7,043 Customers
* Domain: Telecom
* Key Features:

  * Customer Demographics
  * Contract Type
  * Payment Method
  * Internet Service
  * Monthly Charges
  * Tenure
  * Churn Status

## Key Insights

* Month-to-Month contracts contributed 1,655 churned customers.
* Electronic Check customers showed the highest churn rate (45%).
* Fiber Optic customers accounted for 69.4% of total churn.
* Customers in their first 12 months showed elevated churn risk.

## Retention Recommendations

* Encourage AutoPay adoption.
* Improve Fiber Optic service quality.
* Launch loyalty programs for new customers.
* Focus retention campaigns on Month-to-Month customers.
